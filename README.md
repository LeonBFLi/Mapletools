# Mapletools

## Win10 红点监控自动最小化工具

文件：`win10_red_monitor_sleep.py`

### 功能
- 可视化框选屏幕监控区域。
- 监控该区域内红色像素比例变化（用于识别“红点在跳动”的场景）。
- 触发后自动最小化当前前台窗口。

### 运行环境
- Windows 10
- Python 3.9+

### 从仓库下载
```bash
git clone <你的仓库地址>
cd Mapletools
```

> 如果你是直接下载 ZIP，解压后进入目录即可。

### 安装依赖
```bash
pip install pillow
```

### 运行
```bash
python win10_red_monitor_sleep.py
```

> 如果你是 **双击 `.py` 文件** 启动，建议先确认已安装依赖（`pip install pillow`）。  
> 当前版本在缺少依赖或启动失败时会弹窗并在 CMD 中等待回车，不会再“秒退”。
>
> 也可以直接双击仓库里的 `START.bat` 启动（推荐），它会自动寻找 `py -3` / `python` 并在异常退出时保留窗口。

### 是否需要编译？
- **不需要编译**，该工具是 Python 脚本，安装依赖后可直接运行。
- 可选：如果你想打包成 exe，再使用 `pyinstaller` 等工具进行打包。

### 使用步骤
1. 点击 **选择监控区域**，拖拽选中你要监控的小区域。
2. 按需调整阈值参数（默认值通常可直接使用）。
3. 点击 **开始监控**。
4. 当区域内红色像素出现明显跳动时，程序会自动最小化当前前台窗口。

### 参数说明
- **红色阈值 (R >=)**：红色通道最低值。
- **红色优势 (R-G/B >=)**：红色相对绿/蓝至少高出的数值。
- **红点比例阈值**：相邻两次检测的红点比例变化达到该值则视为“跳动”。
- **检测间隔(ms)**：采样频率。
- **触发冷却(秒)**：防止连续重复触发。

### 注意事项
- 该脚本使用 `ImageGrab` 截屏，请在本机显示环境下运行。
- 远程桌面、缩放、HDR 等显示设置可能影响颜色识别效果。
- 触发动作会最小化当前前台窗口，请谨慎测试（可先用“立即测试最小化”按钮验证）。

### 常见问题：双击后 CMD 一闪而过
- **现象**：双击脚本后 CMD 弹出又立刻关闭，看不到“选择监控区域”窗口。
- **原因**：通常是启动阶段报错（最常见是缺少 `Pillow` 依赖），进程立即退出导致窗口看起来“秒退”。
- **解决**：
  1. 在命令行先安装依赖：`pip install pillow`
  2. 建议用命令行启动以便查看日志：`python win10_red_monitor_sleep.py`
  3. 当前版本已在启动失败时增加错误提示和“按回车退出”，方便定位问题。
