# Mapletools

## Win10 红点监控自动最小化工具

文件：`win10_red_monitor_sleep.py`

### 功能
- 可视化框选屏幕监控区域。
- 监控该区域内“明显偏红”的像素是否出现。
- 触发后自动最小化当前活动窗口。
- 每次触发动作都会对目标区域截图，并保存到脚本所在目录。
- 触发最小化后，默认 20 秒自动还原最近最小化窗口并复检；若仍有红点则继续最小化并循环，直到还原后红点消失。
- UI 显示系统日期时间和运行日志。

### 运行环境
- Windows 10
- Python 3.9+

### 从仓库下载
```bash
git clone <你的仓库地址>
cd Mapletools
```

> 如果你是直接下载 ZIP，解压后进入目录即可。

### 安装依赖
```bash
pip install pillow
```

### 运行
```bash
python win10_red_monitor_sleep.py
```

### 使用步骤
1. 点击 **选择监控区域**，拖拽选中你要监控的小区域。
2. 按需调整阈值参数（默认值通常可直接使用）。
3. 点击 **开始监控**。
4. 当区域内检测到红点时，程序会自动保存截图并最小化当前活动窗口。
5. 程序每隔“还原延时(秒)”自动还原并复检；若仍有红点则继续截图+最小化，直到某次还原后红点消失为止。

### 参数说明
- **红色阈值 (R >=)**：红色通道最低值。
- **红色优势 (R-G/B >=)**：红色相对绿/蓝至少高出的数值。
- **最少红像素数量**：达到该数量才触发动作。
- **检测间隔(ms)**：采样频率。
- **触发冷却(秒)**：防止连续重复触发。
- **还原延时(秒)**：触发最小化后，自动还原并复检的等待时长（默认 20 秒）。

### 注意事项
- 该脚本使用 `ImageGrab` 截屏，请在本机显示环境下运行。
- 远程桌面、缩放、HDR 等显示设置可能影响颜色识别效果。
- 触发动作会最小化当前活动窗口，请在非关键场景先测试。
