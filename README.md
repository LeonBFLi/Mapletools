# Mapletools

## Win10 红点监控自动休眠工具

文件：`win10_red_monitor_sleep.py`

### 功能
- 可视化框选屏幕监控区域。
- 监控该区域内红色像素比例变化（用于识别“红点在跳动”的场景）。
- 触发后自动发送 Win10 休眠键（`VK_SLEEP`），并尝试关闭显示器。

### 运行环境
- Windows 10
- Python 3.9+

### 安装依赖
```bash
pip install pillow
```

### 运行
```bash
python win10_red_monitor_sleep.py
```

### 使用步骤
1. 点击 **选择监控区域**，拖拽选中你要监控的小区域。
2. 按需调整阈值参数（默认值通常可直接使用）。
3. 点击 **开始监控**。
4. 当区域内红色像素出现明显跳动时，程序会自动触发休眠。

### 参数说明
- **红色阈值 (R >=)**：红色通道最低值。
- **红色优势 (R-G/B >=)**：红色相对绿/蓝至少高出的数值。
- **红点比例阈值**：相邻两次检测的红点比例变化达到该值则视为“跳动”。
- **检测间隔(ms)**：采样频率。
- **触发冷却(秒)**：防止连续重复触发。

### 注意事项
- 该脚本使用 `ImageGrab` 截屏，请在本机显示环境下运行。
- 远程桌面、缩放、HDR 等显示设置可能影响颜色识别效果。
- 触发动作包含系统休眠键，请谨慎测试（可先用“立即测试休眠”按钮验证）。
